# Technical Documentation

CalledIt is a Telegram bot that turns football predictions into private, friend-to-friend exact-score pools.
A group admin creates a pool, players lock one score before kickoff, and a live leaderboard follows the match.
TxLINE provides the real fixture, kickoff time, live score updates, and a cryptographic proof of the final score.
CalledIt's Solana program verifies that proof on-chain before settling the pool.
